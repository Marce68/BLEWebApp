To update from an old major version E1.3.xxx you need to update the boot loader, this require the use of an intermediate major version E1.4.000

Step 1 - Check current firmware in your machine, if E1.3.xxx or S1.3.xxx you need to update to E1.4.000

Step 2 - Update the machine to E1.4.000.bin firmware version

Step 3 - Update the machine to the Bootloader_v2.bin version

Step 4 - Update the machine to M2.6.000 (Milano Pro) or S2.6.000 (Milano Classic)
