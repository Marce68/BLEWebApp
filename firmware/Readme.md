To update from an old major version E1.3.xxx you need to update the boot loader, this require the use of an intermediate major version E1.4.xxx
Step 1 - Check current firmware in your machine, if E1.3.xxx or S1.3.xxx you need to update to E1.4.xxx
Step 2 - Update the machine to E1.4.xxx version
Step 3 - Update the BootLoader
